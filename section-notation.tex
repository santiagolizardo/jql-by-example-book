\chapter{Notation}

This chapter describes the notation used througout the book so that the reader can associate code snippets to concepts more easily.

\section{Code snippets}

\begin{itemize}
    \item Functions\index{Functions} are shown in orange. E.g. \textcolor{orange}{currentUser}
    \item Keywords\index{Keywords} are shown in blue. E.g. \textcolor{blue}{NOT IN}
    \item Strings\index{Strings} are shown in red. E.g. \textcolor{red}{"Create new class"}
\end{itemize}
